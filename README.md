# CodeAlpha_SimpleEcommerce

SnapKart is an E-Commerce Web Application developed as part of the CodeAlpha Full Stack Development Internship.

## Features

* Product Listing
* Product Details Page
* Shopping Cart
* User Login
* User Registration
* Product Search
* Dark Mode

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* EJS

## Installation

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node server.js
```

4. Open browser

```text
http://localhost:3000
```
